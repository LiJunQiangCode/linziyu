# 给姿妤的浪漫惊喜页面

这是一个为林姿妤准备的浪漫惊喜页面，可以上传到GitHub Pages。页面采用苹果风格设计，包含互动元素、背景音乐、动态装饰和轮播照片功能。

## 使用说明

### 1. 上传照片

请将7张照片上传到 `photos` 文件夹中，并按照以下命名规则重命名：
- 1.jpg
- 2.jpg
- 3.jpg
- 4.jpg
- 5.jpg
- 6.jpg
- 7.jpg

### 2. 上传背景音乐

请将背景音乐文件上传到根目录，并命名为 `music.mp3`。

### 3. 部署到GitHub Pages

1. 在GitHub上创建一个新的仓库
2. 将此目录中的所有文件上传到仓库
3. 在仓库设置中启用GitHub Pages

## 页面功能

- **欢迎页面**：用户需要点击按钮才能进入主页面
- **苹果风格动态装饰**：页面中有彩色气球和透明泡泡漂浮，更逼真、更美观
- **轮播照片**：中间区域自动轮播展示照片，带有精美的渐影效果
- **指示器导航**：可以通过底部指示器手动切换照片
- **键盘控制**：支持左右箭头键控制照片切换
- **环绕祝福消息**：照片周围不断弹出浪漫的祝福话语，更紧密、更生动
- **背景音乐**：进入主页面后自动播放背景音乐

## 设计特点

- **苹果风格UI**：气球和泡泡效果模仿苹果输入法的设计风格
- **渐影照片展示**：照片边缘带有精美的阴影效果
- **紧密的消息弹窗**：祝福消息出现更频繁，分布在照片四周
- **响应式设计**：适配不同屏幕尺寸，在手机上也有良好体验
- **平滑动画**：气球、泡泡和消息都有自然流畅的动画效果

## 注意事项

- 请确保照片尺寸适中，过大的照片可能会导致加载缓慢
- 音乐文件建议使用MP3格式，大小尽量控制在几MB以内
- 由于浏览器安全限制，背景音乐可能需要用户交互后才能播放
- 可以根据需要修改 `script.js` 中的爱情宣言内容
